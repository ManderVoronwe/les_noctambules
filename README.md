## Prérecquis

Node 21.0

## Installation

1. Cloner le repo
   ```sh
   git clone https://github.com/ManderVoronwe/les_noctambules.git
   ```
2. cd

   ```sh
   cd les_noctambules
   ```

3. Installer les dépendances
   ```sh
   npm i
   ```

## Usage

Lancer le serveur ainsi :

```sh
npm start
```
